# Speak-Easy

Speak-Easy is a web application designed to help users improve their public speaking skills through interactive practice, feedback, and analytics.

## Features
- Practice speeches with real-time feedback
- Analyze speech patterns and filler words
- Track progress over time
- User-friendly interface

## Getting Started

### Prerequisites
- Node.js (v14 or higher recommended)
- npm (v6 or higher)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/shireen-vir/Speak-Easy.git
   cd speak-easy
   ```
2. Install dependencies:
   ```bash
   npm install
   ```

### Running the App
To start the development server:
```bash
npm start
```
Open [http://localhost:3000](http://localhost:3000) in your browser.

### Running Tests
```bash
npm test
```

### Building for Production
```bash
npm run build
```
The production-ready files will be in the `build` folder.
